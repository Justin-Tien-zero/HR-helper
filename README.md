# HR-helper

This is a React + Vite application.

## 🚀 起步 (Run Locally)

**Prerequisites:** Node.js v20+

1. 安裝套件：
   ```bash
   npm install
   ```
2. 設定環境變數：
   將 `.env.example` 複製為 `.env.local` 並填入你的 `GEMINI_API_KEY`。
3. 啟動本機伺服器：
   ```bash
   npm run dev
   ```

## 📦 部署 (Deployment)

專案已設定好 GitHub Actions，當你推播上 `main` 或 `master` 分支時，會自動部署到 [GitHub Pages](https://Justin-Tien-zero.github.io/HR-helper/)。請記得到 GitHub 倉庫的 **Settings > Pages** 將來源設定為 **GitHub Actions**。
